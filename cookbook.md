# ID variables
##############
1 - subject : The ID of the volunteer of the test, values in range [1:30]
2 - activity : The activity which is performed when doing the test and taking the corresponding measurements,
activity labels : 
					1 WALKING
					2 WALKING_UPSTAIRS
					3 WALKING_DOWNSTAIRS
					4 SITTING
					5 STANDING
					6 LAYING
